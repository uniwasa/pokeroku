# pokeroku

ポケモンの図鑑機能とパーティ管理機能 を搭載したアプリをFlutterで制作。

状態管理にはRiverpod、Freezedなどを使用してMVVMアーキテクチャで開発。

パーティ管理機能やログイン機能にはFirebaseを使用。
